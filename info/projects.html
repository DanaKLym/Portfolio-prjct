Weather app

The Weather app was built using HTML, CSS and Vanilla JS.

It is designed with the dynamic background which changes according to the part of the day and dynamic animation for a better representation of weather conditions in the region.

The information about the weather is taken from the OpenWeatherMap API. With the help of Bootstrap 5 the app can be used on mobile devices.

The website is hosted on Netlify platform and is open-sourced on GitHub.

Carpathian Mountains app

Take a look at the Carpathian Mountains landing page, built with HTML, CSS and Vanilla JS.

The project includes short information about Ukrainian Carpathians, providing the examples of places to visit and offers interactive activity to the user at the bottom of the page.

The website is hosted on Netlify platform and is open-sourced on GitHub.
